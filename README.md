# 📊 SQL Fundamentals 🚀

A beginner-friendly repository to learn **SQL from basics to advanced concepts**, specially useful for **DBMS lab exams, viva, and interview preparation**.

---

## 📌 What You’ll Learn

* 🧱 Basic SQL Queries
* 🔍 SELECT, WHERE, ORDER BY
* 🔗 JOIN (INNER, LEFT, RIGHT, FULL)
* 🧮 Aggregate Functions (COUNT, SUM, AVG)
* 📊 GROUP BY & HAVING
* 🧩 Subqueries
* 🛠 Views & Indexes
* ⚙️ Triggers & Procedures
