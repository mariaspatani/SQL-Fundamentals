## 🔥 Topics Covered

### 🧱 Basics

* CREATE, INSERT, SELECT
* UPDATE, DELETE
* Data Types

### 🔗 Joins

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN

### 🧠 Advanced

* Subqueries
* Views
* Triggers
* Stored Procedures

### ⚙️ SQL Commands Classification

* DDL (CREATE, ALTER, DROP)
* DML (INSERT, UPDATE, DELETE)
* DCL (GRANT, REVOKE)
* TCL (COMMIT, ROLLBACK)
